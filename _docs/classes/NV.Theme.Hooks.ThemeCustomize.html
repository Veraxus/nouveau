register()

register(\WP_Customize_Manager  $wp_customize)

This hooks into 'customize_register' (available as of WP 3.4) and allows you to add new sections and controls to the Theme Customize screen.

Note: To enable instant preview, we have to actually write a bit of custom javascript.

generate_css()

generate_css(string  $selector, string  $style, string  $mod_name, string  $prefix = '', string  $postfix = '', boolean  $echo = true) : string

This will generate a line of CSS for use in header output. If the setting ($mod_name) has no defined value, the CSS will not be output.

Parameters

string $selector

CSS selector
string $style

The name of the CSS property to modify
string $mod_name

The name of the 'theme_mod' option to fetch
string $prefix

Optional. Anything that needs to be output before the CSS property
string $postfix

Optional. Anything that needs to be output after the CSS property
boolean $echo

Optional. Whether to print directly to the page (default: true).

Returns

string —

Returns a single line of CSS with selectors and a property.
